Emmet abbreviation 
1. Emmet is just like a prediction.
2. It will predict what you want to use.
For Example:-if you will use exclamatory mark (!), And hit enter then visual studio code automatically print the structure of the HTML document.

Note:- you can save the HTML document with the .htm and .html extension but  sometimes if you save your HTML document with .htm then it can give error on live server.
So, the best practice is that to save your HTML document with .htm extension.

Html Structure(Boilerplate template)

	<!DOCTYPE html>
	<!-- It defines the type of the html document -->
	<html lang="en">
	<!-- This is the starting tag of html document -->
	<!-- lang is the attribute which define the default language of the web page 
	     or you can say that it declare the language of the web page. -->
	<head>
	    <meta charset="UTF-8">
	    <!-- meta means data about data -->
	    <!-- meta tags are not visible on browsers -->
	    <!-- these tags are used in seo(search engine optimization) -->
	    <!-- we define keywords with the help of meta tag which will help our website to be on top in seo and we can also define the author, summary information  -->
	    <meta http-equiv="X-UA-Compatible" content="IE=edge">
	    <meta name="viewport" content="width=device-width, initial-scale=1.0">
	    <!-- viewport is the visible area of the web page -->
	    <!-- it is different in mobile to mobile, desktop to desktop, etc. -->
	    <title>Document</title>
	    <!-- This defines the title of the web page. -->
	</head>
	<body>
	    <!-- body tag define the content -->
	    <!-- This contains all the information which you want to show to the user -->
	    <h1> Hello World! </h1>
	</body>
</html>
